# mainnet-hth-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面内容不针对任何特定域名或网站，仅作为静态资源存档，供需要时参考或使用。

仓库以简洁、通用为原则，不包含动态后端或数据库依赖，所有页面均为纯前端静态 HTML 文件。

## 目录说明

```
mainnet-hth-resource/
├── docs/               # 存放 HTML 页面及附属资源（CSS、JS、图片等）
├── README.md           # 本文件
```

- `docs/` 目录为默认的静态资源根目录，可直接通过 GitHub Pages 等静态托管服务发布。
- 页面文件命名建议使用小写字母、数字和连字符，避免特殊字符。

## 页面归档说明

- 所有 HTML 页面均为独立完整文件，原则上不依赖外部资源（除 CDN 公共库外）。
- 页面内容与结构保持简洁，避免冗余样式或脚本。
- 归档目的为长期保存与参考，不保证实时更新或适配所有浏览器版本。

## 维护说明

- 本仓库为静态资源存档，不接受功能请求或内容修改建议。
- 如发现页面显示异常或链接失效，欢迎提交 Issue 说明。
- 仓库维护者将定期检查资源完整性，但不对第三方使用或二次分发负责。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `docs/` 目录下的 HTML 文件即可查看。
3. 若需部署到 Web 服务器，将 `docs/` 目录设为根目录即可。
